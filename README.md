\# Blender Speedrun Timer



A simple Speedrun timer addon for Blender.



\## Usage

* Enable Addon
* Save your scene (otherwise the run start button will do nothing except outputting a warning in the console)
* Open "N" Panel
* Go to Blender Speedrun
* Click Start Run (it will clear the scene and start the timer on the bottom left)
* Stop Stops the timer and displays it
* Reset clears the timer and returns to starting scene



\## Features



* Start/stop timer and reset scene to starting scene.
* Displays timer in viewport





\## Installation



1\. Download the latest release ZIP

2\. Open Blender

3\. Edit → Preferences → Add-ons → Install from disk

4\. Select the ZIP file
